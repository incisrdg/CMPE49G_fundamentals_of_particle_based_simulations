# Buffon's Needle Problem

Buffon's Needle problem, one of the earliest problems in geometric probability, was first proposed in the first half of the 18th century. The problem follows like this: Suppose we have a floor made of parallel strips of wood, each the same width, and we drop a needle onto the floor. What is the probability that the needle will lie across a line between two strips?
